About the App

Survello is an application made for COMP229

Survello provides users with the opportunity to multiple surveys. If a user is logged in, it allows the creation of surveys if a user is logged.
If a user is not registered they will be directed to login and in the case of having no credentials, will be then directed to log in.
The surveys are connected to a database that is hosted on MongoAtlas. It is built using node, angular and the sweat and tears of its developers.
